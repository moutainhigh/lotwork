【问题】充值流程：

  • 第一步：进入平台首页，点击【充值】

  • 第二步： 选择您需要充值的方式与银行后，输入金额（如工行在线充值），点击【下一步】

  • 第三步：获取到充值信息后，点击【登录网上银汇款】，请您使用“复制粘贴功能”，避免输入错误信息导致延迟入账。

【问题】提现须知：

  • 1. 提款前请先设置"安全问题"和"安全密码"，并且绑定好银行卡。

  • 2. 提款金额必须在可提现额度范围以内。

  • 3. 当发起提款申请成功后，提款15分钟内到账，如因网银系统问题或其他不可抗力因素影响，到账时间将会延迟，您有任何关于提款问题也可以咨询在线客服

【问题】平台提款安全吗？

  • 通过提交提款申请确保只能由用户自己提款；通过将用户提交的提款银行卡信息与用户注册时提交的身份认证信息进行核对，确保用户只能提款到本人银行卡，任何人都无法盗取用户的资金，保证提款的绝对安全

【问题】平台提款流程：

  • 第一步：进入平台首页点击【提现】

  • 第二步：选择您需要提款的银行卡，并输入您欲提款的金额后，点击【申请提现】

  • 第三步：请再次核实您欲提现的银行卡，并输入"安全问题"和"安全密码"提交即可"

  • 第四步：申请提款成功后，您提款的金额将会处于冻结状态

  • 第五步：您可通过【用户名快捷导航】→“账户明细”→“提现记录”查看您的提款状态。状态显示“提款成功”后，请登陆银行确认到账。

【问题】点击【申请提现】后没有反应？

    当您进入提现页面，点击【申请提现】按键却没有反应时：

  • 1.请检查您的『可提现额度』是否大于您申请的『提现金额』。

  • 2.请检查您申请的『提现金额』是否大于100元 。

  • 3.请查询您当日的提现次数，是否已经超过次数。

    • 如以上3点未能协助您，请您联系在线客服。

【问题】绑定银行卡真实姓名和卡号填写有误有什么影响？填错了有什么办法补救吗?

  • 如果注册时没有如实填写绑定银行卡真实姓名和卡号，所绑定信息与您银行卡真实信息不一致时，您的提款将不予处理。因此绑定银行卡时，必须仔细填写确保无误！当遇到了绑定错误银行卡信息等此类相关问题，为了您的资金安全，请您与平台在线客服联系，客服人员核实您的身份后将协助您处理

【问题】平台提现限额、手续费及时间?

  • 用户提现产生的手续费由平台吸收，会员不需支付任何手续费，所以为了保证出款效率，避免恶意提款给平台带来的消极影响，目前使用“向平台提现”的方式提款，平台设置最低提款金额是100元，每天限提2次，无最高提现金额限制。全天24小时您随时都可以进行提款，您的提现会再发起提现后的15分钟内到账，如因网银系统问题或其他不可抗力因素影响，到账时间将会延迟。

【问题】充值后是否可以立即提款？

  • 由于提款会扣除手续费，这样会使您充值的现金贬值；另外为了防止洗钱行为，充值后需进行游戏达到“可提现额度”方能进行提现。
